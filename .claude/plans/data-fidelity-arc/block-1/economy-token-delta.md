## Method

- **Baseline** (un-capped): the cold `tools::*::handle` output at
`verbosity:detailed`, `limit:u32::MAX`, no cursor — the whole result.
- **Default** (economy): the same handler at `verbosity:concise` and the
default page cap (50), no cursor — the first page a caller gets.
- **Token proxy**: `tokens = bytes / 4` (the `wire` projection is
`serde_json::to_string` into one text block); raw bytes reported
alongside so the relative delta does not hinge on the divisor.
- `co_change` is measured at low thresholds (the worst pre-block
offender); the diff-aware tools run over a representative single-file
changeset.
