## Method

- **Baseline** (reflexive, no Ariadne): Σ bytes of `grep` matching-line text for the plain
query across all indexed source files, plus a whole-file `Read` of the file defining the
target symbol. Models the line-range-unaware reflexive path D11 targets; matching-line text
only (no `file:line:` prefix), so the baseline is conservatively understated.
- **Prototype** (search + span): JSON-serialised symbol search hits in the `SymbolSummary`
shape (name, kind, file, 1-based line range), capped at 64, plus the target's
exact span widened by ±3 context lines (D9 `context(±N)`).
- **Token proxy**: `tokens = bytes / 4` (OpenAI English rule of thumb); raw bytes reported
alongside so the verdict does not hinge on the divisor — the gate is a relative delta whose
scaling cancels.
- Both arms operate over the same indexed-file corpus; targets are asserted to resolve.
