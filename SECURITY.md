# Security Policy

## Supported Versions

Security fixes land on the latest minor release line only. Older lines receive
no backports — upgrade to the latest `1.x` to stay supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do **not** open a public issue for security vulnerabilities.

Report privately through GitHub's
[private vulnerability reporting](https://github.com/mind-decay/ariadne/security/advisories/new)
(repository **Security** tab → **Report a vulnerability**). If you cannot use
GitHub, contact the maintainer at `<OWNER-CONTACT-PLACEHOLDER>`.

Include the affected version (`ariadne --version` or commit hash), a
reproduction, and the impact you observed. You can expect an acknowledgement
within a few days; once a fix ships we will credit you in the advisory unless
you prefer to remain anonymous.
